Instrucciones

El modulo permite asignar una imagen para diferentes contratos y notas de credito a partir de un archivo CSV.
1-.El archivo CSV debera contener como Cabeceras: CONTRATO_PRIN|AFILIADO|EVENTO
2-.Al leer el archivo CSV/TXT el sistema mostrara una tabla con los datos del archivo mas otras columnas.
3-.Se mostrara las columnas de Nombre, Estado y Estatus(campo vacio) para validar si son correctos.
4-.Se activaran los botones Exportar Excel y Cargar Datos.
5-.En Visualizador de Imagenes solo acepta un solo archivo y en formato (JPG,PNG,BMP).
6-.El Boton Cargar Datos comenzara a asignar la imagen a los contratos y a las Notas de Credito que No Existan.
Se motrara los Estatus con(✓) se asigno correctamente. (X) fallo la asignacion porque ya existe la nota de credito 4.
